# Pycle Car Rental Web App

This is a car rental system built with the MERN stack.

##


## Run Locally

### 1. Clone repo

```
$ git clone git@github.com:DavidLivingstoneHini/Pycle-Project.git
$ cd Pycle-Project
```

### 2. Running the Backend

```
$ npm install
$ npm start
```

### 3. Running the Frontend

```
# open your terminal
$ cd frontend
$ npm install
$ npm start
```
